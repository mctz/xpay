微信官方接口开通说明

申请开通流程

1.进入XPay支付页面捐赠成功后您将获取到开通教程 点我立即获取教程

2.根据获取的教程文档逐步完成每一步，成功开通官方接口， 登录 微信支付普通商户平台 中查看， 查看官方接口文档进行下一步开发即可

特别说明

此教程为个人通过正规渠道开通微信官方接口，共需开通成本约600元，全部由第三方平台收取（如微信官方认证费用），本平台XPay不收取任何费用，也不提供任何代开服务。 且开通需要一定时间（如微信官方工作日审核时间）与精力，怕麻烦者可绕道

开发教程

具体请查看 微信支付普通商户官方开发文档 ，有详细的开发与接口说明，向您发送的XPay源码中也包含详细文档和示例，以下仅简要说明主要所需的接口：

1.使用 统一下单API 接口获取每笔订单收款二维码

2.通过 支付结果通知 获取官方通知回调

3.使用 查询订单API 接口查询每笔订单支付结果
